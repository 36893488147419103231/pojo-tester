Tests Preconditions

Our tests will be performed on pojo-classes. Each library has it's own pojo-class in a package name like library name. There are few of them, beacuse we want to try generate pojo-methos using Lombok, Apache's commons lang 3 and Google's guava. We also measure code coverage using JaCoCo 0.7.7.201606060606.

Classes contains fiels as shown below:

public class Pojo_Standard_Generated_Methods {
    private int a;
    private float b;
    private String c;
    private TestEnum d;

    enum TestEnum {
        ONE, TWO
    }

    // standard generated methods
}

POJO-TESTER

POJO-TESTER tests looks like this:

@Test
public void Should_Test_Pojo() {
    // given
    final Class[] classesUnderTest = {Pojo_Guava_Generated_Methods.class,
                                      Pojo_Apache_Generated_Methods.class,
                                      Pojo_Lombok_Generated_Methods.class,
                                      Pojo_Standard_Generated_Methods.class};
    // when

    // then
    assertPojoMethodsForAll(classesUnderTest).areWellImplemented();
}
